Вопрос 25

Составьте словесный алгоритм для перехода дороги на светофоре.

Структурированное описание алгоритма

Цикл с предусловием

1. Начало
2. Положить flag = true
3. Подойти к переходу
4. Пока flag равен true
   4.1. Посмотреть на свет пешеходного светофора
   4.2. Если горит зелёный
        то
        4.2.1. Перейти дорогу
        4.2.2. flag = false
5. Конец

Цикл с постусловием

1. Начало
2. Положить flag = true
3. Подойти к переходу
4. Повторить
   4.1. Посмотреть на свет пешеходного светофора
   4.2. Если горит зелёный
        то
        4.2.1. Перейти дорогу
        4.2.2. flag = false
   пока flag равен true
5. Конец
